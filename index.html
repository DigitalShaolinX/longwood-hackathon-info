Get Ready

How to Prepare

You don't need to be an expert — just come ready to learn and build!

Python Track

Since AI and data science work is done primarily in Python, even a foundational understanding will make a huge difference during the 6-hour event.

  • Learn basic syntax & variables
  • Understand loops & functions
  • Practice small coding exercises

Free Recommended Course:
kaggle.com/learn/python

Web Track

Students who aren't comfortable with Python can tackle HTML & web-based coding challenges instead.

  • Build your project's presentation layer
  • Use HTML, CSS & JavaScript
  • A website makes a great front-end for any AI project!

Tip: This is important since all students should consider building a simple HTML/JS site as the visual layer for their solution!

Before You Go

  • Sign & return the photo release form
  • Note your bus departure time
  • Bring a charged laptop if you have one
